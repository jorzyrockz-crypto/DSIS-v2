Help & Documentation

Quick operator guide for daily DSIS workflows.

1. Getting Started

  • Login with your School ID and profile.
  • Use top navigation: Dashboard, Inventory, Action Center, Archives.
  • Open Profile menu for settings, appearance, updates, and sign out.

2. Inventory Workflow

  • Create or edit records in Manage Inventory.
  • Fill agency/signatory fields and add item rows.
  • Finalize data to save record into the records table.

3. Action Center & Archives

  • Review EUL and inspection status in Action Center.
  • Archive eligible items and prepare Waste Materials Report entries.
  • Use Archives for historical records and WMR print actions.

4. Data & Security

  • Use Data Hub for ICS/PAR import/export with validation preview.
  • Use Profile > Security for trace checks and recovery actions.
  • Use Audit Trail to review activity logs with filters and exports.

5. Keyboard Shortcuts

  • Ctrl+K: Search
  • Alt+1..4: Switch main views
  • Alt+D: Data Manager
  • Esc: Close active overlay/modal
